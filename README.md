> 一款基于分级记忆法的单词学习 Web 应用，支持四级分类、自定义复习配比、单词本导入及本地数据持久化。

---

## ✨ 主要功能

| 功能 | 描述 |
|------|------|
| **📇 单词卡片** | 正面显示单词、音标、例句；点击翻转显示释义和例句翻译，支持 3D 翻转动画。 |
| **🎯 四级分类** | 不认识 / 不太熟 / 认识 / 完全熟，点击按钮即移入对应词库，并自动跳转下一个单词。 |
| **🔍 词库筛选** | 可单独复习任一等级词库，或使用“总体记忆”按配比随机抽取单词。 |
| **⚖️ 总体记忆配比** | 自定义“不认识 : 不太熟 : 认识”的权重，智能决定下次复习哪个单词。 |
| **📋 单词列表** | 展开查看当前词库所有单词，点击任意单词可快速跳转，方便回溯修正。 |
| **📊 学习统计** | 实时显示各等级单词数量，掌握情况一目了然。 |
| **📂 单词本导入** | 支持导入 JSON 格式的单词本，替换当前所有单词，数据自动保存。 |
| **💾 数据持久化** | 单词等级、配比设置自动保存在浏览器 localStorage，刷新不丢失。 |

## 📖 使用指南

### 基础操作
1. **学习单词**：卡片正面显示单词、音标、例句，点击卡片翻转查看释义和例句翻译。
2. **标记等级**：点击下方四个按钮（不认识 / 不太熟 / 认识 / 完全熟），单词会移入对应词库并自动跳转下一个。
3. **切换单词**：使用“上一个 / 下一个”按钮或单词列表快速跳转。
4. **切换词库**：点击筛选栏按钮，可选择只复习某个等级或使用“总体记忆”。

### 总体记忆配比设置
- 在“总体记忆”模式下，会显示配比调整区域。
- 可设置“不认识 : 不太熟 : 认识”的权重（如 5:3:2），程序将按比例随机抽取单词复习。
- 配比修改后自动保存，刷新页面不丢失。

### 单词列表
- 点击“查看单词列表”按钮，展开当前词库的所有单词。
- 当前单词高亮显示，点击任意单词可快速跳转，方便修正标记。

### 导入单词本
1. 准备一个 JSON 文件，格式如下：
   ```json
   [
     {
       "word": "example",
       "phonetic": "/ɪɡˈzæm.pəl/",
       "meaning": "例子",
       "example": "This is an example.",
       "exampleTranslation": "这是一个例子。",
       "level": "unfamiliar"
     }
   ]
   ```
2. 点击“导入单词本”按钮，选择 JSON 文件。
3. 导入成功后，页面自动刷新显示新单词，数据自动保存。
4. 刷新或关闭浏览器后，导入的单词仍然存在。

### 数据持久化
- 所有单词等级和配比设置都会自动保存到浏览器中。
- 清除浏览器数据会重置所有单词，回到初始状态。
