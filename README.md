# 📖 单词卡片

一款基于分级记忆法的单词学习 Web 应用。支持四级分类（不认识/不太熟/认识/完全熟）、词库筛选、总体记忆配比随机抽取、单词本导入及本地数据持久化。

## ✨ 主要功能

- **单词卡片**：正面显示单词、音标、例句；点击翻转显示释义和例句翻译。
- **四级掌握度**：点击按钮将单词移入对应词库，自动跳转下一个。
- **词库筛选**：可单独复习“不认识”“不太熟”“认识”“完全熟”词库，或使用“总体记忆”模式。
- **总体记忆配比**：自定义“不认识:不太熟:认识”的复习权重，按比例随机抽取单词。
- **单词列表**：可展开当前词库单词列表，点击任意单词跳转，方便回溯修正。
- **学习统计**：实时显示各等级单词数量。
- **单词本导入**：支持导入 JSON 格式的单词本（替换当前所有单词）。
- **数据持久化**：单词等级、配比设置自动保存在浏览器本地，刷新不丢失。

## 🚀 如何运行

1. 下载或克隆本项目到本地。
2. 直接用浏览器打开 `index.html` 即可开始使用（无需安装任何依赖）。

## 📖 使用指南

### 基本操作
- **学习单词**：卡片正面显示单词、音标、例句，点击卡片翻转查看释义和例句翻译。
- **标记等级**：点击下方四个颜色按钮（不认识、不太熟、认识、完全熟），单词会被移入对应词库，并自动跳转下一个。
- **切换单词**：使用“上一个”“下一个”按钮在当前词库内切换（非总体记忆模式按顺序，总体记忆模式随机）。
- **切换词库**：点击筛选栏按钮，可选择只复习某个等级或使用“总体记忆”。

### 总体记忆配比设置
- 在“总体记忆”模式下，会显示配比调整区域。可设置“不认识:不太熟:认识”的权重，程序将按比例随机抽取单词。

### 单词列表
- 点击“查看单词列表”按钮，可展开当前词库所有单词，点击单词可快速跳转。

  {
